# Metacognitive Orchestration in Multi-Agent Systems

This proposal extends the **MetaCog-Bench** framework — submitted to the DeepMind / Kaggle *Measuring Progress Toward AGI — Cognitive Abilities* Hackathon 2026, Metacognition track — from single-model stated-vs-behavioral metacognition to **population-level meta-learning in open-ended multi-agent environments** (Minecraft-grounded).

**Relation to MetaCog-Bench submission**
- MetaCog-Bench asks: *Does an individual model self-verify when offered a tool?*
- This proposal asks: *Does a multi-agent system self-correct at the population level through metacognitive orchestration?*

**Author:** Shivdeep Singh, Independent AI Researcher, India
